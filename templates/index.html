💰

비용 최적화

부서별 예산 한도 설정, 사용량 알림, 모델 자동 전환으로 AI 비용을 절감합니다.

민감 정보 탐지, 프롬프트 필터링, 출력물 감사 로그로 데이터를 보호합니다.

신규 AI 도구 도입, 고비용 모델 사용, 외부 API 연동을 승인 절차로 관리합니다.

이용 방법

4단계로 시작하세요

복잡한 설정 없이 빠르게 AI 거버넌스 체계를 구축할 수 있습니다.

01

SSO 연동

사내 SSO(Okta, Azure AD 등)와 연동하여 임직원 계정을 자동 동기화합니다.

02

정책 설정

부서별 AI 사용 정책, 허용 모델, 예산 한도를 관리자 콘솔에서 설정합니다.

03

프록시 배포

AI Hub 프록시를 통해 모든 AI API 호출을 중앙에서 라우팅하고 기록합니다.

04

모니터링

대시보드에서 사용량·비용·보안 이벤트를 실시간으로 확인하고 대응합니다.

AI Hub 도입 전 궁금한 점을 확인하세요.

GPT-4o, Claude 3.5, Gemini Pro, Llama 3 등 주요 LLM을 지원합니다. 사내 자체 호스팅 모델(vLLM, Ollama)도 프록시를 통해 연동할 수 있습니다.

네. AI Hub 프록시 또는 브라우저 확장 프로그램을 통해 기존 도구 사용을 중앙에서 모니터링하고 정책을 적용할 수 있습니다.

클라우드 SaaS 버전은 SSO 연동 후 1~2일 내 운영 가능합니다. 온프레미스 배포는 보안 검토 포함 2~4주가 소요됩니다.

SaaS 버전은 국내 AWS 리전(seoul)에 저장됩니다. Enterprise 플랜은 완전 온프레미스 배포로 모든 데이터가 사내 인프라에만 존재합니다.
